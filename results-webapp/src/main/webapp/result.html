BPMN MIWG Automated test results - Proof of concept

There are two hand-produced test examples below. Both use a base HTML structure enhanced with data-* attributes when mi.init() is called. To declare this object link to the bpmn-miwg.js in the js subfolder.

  • Example for XPath Test Runner to emulate
  • Example for XML Compare to emulate

The following is a (hopefully complete) list of enhancements to make the previous (xml-compare) html work as the above examples:

  • Add link elements to head
  • Add script elements to end
  • Add backToFindings named anchor and associated div
  • Add mi.init() script
  • Add container class to the testresults div
  • Add well and span12 to class alongside test, also add data-test="X.y.z"
  • Add spans with attributes data-frag='text' or data-xpath='xpath' as child nodes of the p child of the div with class="finding"
  • Add data-test='x.y.z' to the div with class='test'
